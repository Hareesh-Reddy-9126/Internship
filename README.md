## Chat Application (level3/task1)
- Run server first:
  - `cd level3/task1`
  - `javac Server.java`
  - `java Server`
- Then run client in another terminal:
  - `cd level3/task1`
  - `javac Client.java`
  - `java Client`
- Type messages; use `exit` to close.
